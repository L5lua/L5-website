# Get started

L5 is an open source library for creative coding in Lua. Create algorithmic and interactive artwork with code on the computer. L5 is built with Love2d and Lua as its software stack. To begin, you must install Love2d and then download our L5 Getting Started directory.

DIAGRAM GOES HERE showing Love2d + your folder with L5.lua and main.lua

## Overview

To get started using L5 on your computer, you need to complete the following steps:

1. Download Love2d
2. Download the L5-Starter folder and extract it (uncompress the folder).
3. Open and edit the `main.lua` file inside the L5-Starter folder.
4. Drag the L5-starter folder on top of the Love2d file to run your program.
